## Описание функционала
Приложение представляет собой агрегатор новостей с поддержкой следующих функций:

Динамическая загрузка: Новости подгружаются из внешнего API.

Фильтрация: Возможность выбора категории (Бизнес, Технологии, Спорт и т.д.).

Поиск: Полнотекстовый поиск по ключевым словам.

Модальное окно: Просмотр краткого содержания статьи без перехода на сторонний ресурс.

История просмотров: Сохранение заголовков последних 5 прочитанных новостей в localStorage.

Пагинация: Кнопка «Загрузить ещё» для подгрузки новых порций данных.
## Использовано GNews API
## Примеры запросов
По категориям: https://gnews.io/api/v4/top-headlines?category=technology&lang=ru&apikey=КЛЮЧ

Поиск по слову: https://gnews.io/api/v4/search?q=космос&lang=ru&apikey=КЛЮЧ
